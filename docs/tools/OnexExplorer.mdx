# OnexExplorer

OnexExplorer is an open-source tool designed for unpacking and repacking **.NOS data files** from the MMORPG **NosTale**. 
It allows users to explore, view, export, replace, and even add their own content to these files, making it a valuable resource for modding and customizing game assets.

## Key Features

- File Compatibility: OnexExplorer supports a wide range of .NOS file types (not all but most of it)
- User-Friendly Interface: Built on Qt 5.5+ with OpenGL, it provides a graphical interface for easier navigation and manipulation of game files.
